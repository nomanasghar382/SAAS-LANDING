# SellPilot AI — LinkedIn Video Guide

## Quick Start (Record in 5 Minutes)

1. Run the app:
   ```bash
   npm run dev
   ```

2. Open in browser (fullscreen):
   ```
   http://localhost:3000/promo
   ```

3. Press **F11** for fullscreen (Windows) or **Ctrl+Cmd+F** (Mac)

4. Screen record with:
   - **Windows:** Win + G (Xbox Game Bar) or OBS Studio
   - **Mac:** QuickTime → New Screen Recording
   - **Free online:** Loom, CapCut desktop

5. Record **one full loop** (~54 seconds) or **two loops** for a 90s video

6. Add background music in CapCut / Premiere (optional but recommended)

---

## Video Specs for LinkedIn

| Setting | Recommended |
|---------|-------------|
| Aspect ratio | **1:1 (square)** or **16:9 (landscape)** |
| Resolution | 1080×1080 or 1920×1080 |
| Length | 30–90 seconds |
| Format | MP4 |
| Captions | Yes (80% watch muted) |

**Crop tip:** Record at 16:9, then crop to 1:1 in CapCut for mobile feed.

## Pro Recording Tips

1. **Hide taskbar** before recording (fullscreen F11)
2. **Close notifications** (Focus Assist on Windows)
3. **Record at 60fps** if possible for smooth animations
4. **Add captions** in CapCut auto-caption feature
5. **First 3 seconds** = hook (don't start with logo fade — trim if needed)
6. **End card** — pause on CTA scene for 2 extra seconds in editor
